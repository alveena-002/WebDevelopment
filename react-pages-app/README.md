# React Router Practice Task

## Description

Is task me maine React Router ka use karke ek simple 3-page website banayi.

Pages:
- Home
- About
- Dashboard

Dashboard ko Protected Route banaya hai. Agar user authenticated ho to Dashboard access kar sakta hai, warna usay Home page par redirect kar diya jata hai.

## Features

- React Router setup
- Home Page
- About Page
- Protected Dashboard Page
- Navigation using Link
- Redirect using Navigate

## Technologies Used

- React
- Vite
- React Router DOM

## What I Learned

Is task me maine seekha:

- React Router install aur setup karna.
- BrowserRouter ka use.
- Routes aur Route banana.
- Link se pages ke darmiyan navigation karna.
- Protected Route create karna.
- Navigate ka use karke unauthorized user ko redirect karna.
- Authentication ka basic concept samajhna.

## How to Run

```bash
npm install
npm run dev
```

Project browser me open hoga:

```
http://localhost:5173
```
