# React + Supabase Users Table

## Description

This project demonstrates how to connect a React application with Supabase. The data is fetched from the `users` table in the Supabase database and displayed in a frontend table using React.

## Features

- Connect React with Supabase
- Fetch data from the `users` table
- Display users in a responsive HTML table
- Use React Hooks (`useState` and `useEffect`)

## Technologies Used

- React (Vite)
- JavaScript
- Supabase
- HTML
- CSS

## Installation

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

## Project Structure

```
src/
├── App.jsx
├── supabase.js
└── main.jsx
```

## Learning Outcome

- Learned how to connect a React application with Supabase.
- Learned how to fetch data from a Supabase database.
- Learned how to use `useEffect` for fetching data when the component loads.
- Learned how to store data using `useState`.
- Learned how to display dynamic data in a table using the `map()` function.
