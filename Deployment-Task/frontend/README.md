# Deployment Task – React Frontend & Express Backend (Vercel)

## 📌 Project Overview

This project demonstrates how to deploy a full-stack application to production using **Vercel**. The frontend is built with **React (Vite)**, and the backend is built with **Node.js** and **Express.js**. Both applications are deployed separately on Vercel and communicate through a live API.

---

## 🚀 Features

* React frontend deployment on Vercel
* Express backend deployment on Vercel Serverless Functions
* REST API endpoint
* CORS enabled
* JSON response handling
* Live production deployment
* GitHub integration with automatic deployments

### Clone Repository


### Frontend

```bash

```
