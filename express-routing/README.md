# Middleware, Error Handling & CORS

## Implemented Features

- Added custom logger middleware.
- Configured CORS.
- Implemented global 404 handler.
- Added centralized error handling middleware.
- Improved API request and response handling.

## How to Run

```bash
npm install
node app.js
```
